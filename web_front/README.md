# GigaFood - Веб-приложение для отслеживания калорий

Веб-версия мобильного приложения GigaFood для отслеживания калорий и питания.

## Структура проекта

```
web/
├── index.html              # Главная страница
├── login.html              # Страница входа
├── register.html           # Страница регистрации
├── password-reset.html     # Страница сброса пароля
├── new-password.html       # Страница создания нового пароля
├── dashboard.html          # Главная страница дашборда
├── menu.html               # Страница меню
├── profile.html            # Страница профиля пользователя
├── recommendations.html    # Страница рекомендаций
├── weekly-report.html      # Еженедельный отчет
├── styles.css              # CSS стили
├── script.js               # JavaScript функциональность
└── readme.txt              # Этот файл
```

## Функциональность

### Страницы авторизации
- **Вход** - форма входа с email и паролем
- **Регистрация** - создание нового аккаунта
- **Сброс пароля** - восстановление доступа к аккаунту
- **Новый пароль** - установка нового пароля

### Основные страницы
- **Дашборд** - главная страница с целью по калориям, прогрессом и кнопками действий
- **Профиль** - настройки пользователя, включая автоматический расчет калорий
- **Рекомендации** - советы по питанию
- **Меню** - навигация по разделам приложения
- **Еженедельный отчет** - статистика за неделю

## Особенности дизайна

- **Адаптивный дизайн** - работает на всех устройствах
- **Современный UI** - чистый и минималистичный интерфейс
- **Цветовая схема** - синий (#007AFF) и зеленый (#34C759) как основные цвета
- **Мобильный подход** - дизайн адаптирован с мобильного приложения

## Технологии

- **HTML5** - семантическая разметка
- **CSS3** - современные стили с Flexbox и Grid
- **JavaScript** - интерактивность и обработка форм
- **LocalStorage** - сохранение данных пользователя

## Запуск

1. Откройте `index.html` в браузере
2. Или запустите локальный сервер:
   ```bash
   # Python 3
   python -m http.server 8000
   
   # Node.js
   npx http-server
   ```

## Функции JavaScript

- Валидация форм
- Автоматический расчет калорий на основе параметров пользователя
- Сохранение данных в localStorage
- Навигация между страницами
- Анимации и интерактивность

## Адаптивность

Приложение адаптировано для:
- Десктопов (1200px+)
- Планшетов (768px - 1199px)
